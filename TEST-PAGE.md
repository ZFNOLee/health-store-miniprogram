## ⚠️ 常见问题

### Q1: 页面显示「🔴 后端未连接」

**原因**：后端服务未启动

**解决**：
```bash
# 检查后端服务
curl http://localhost:3000/api/health

# 如果没响应，重启后端
cd /home/admin/openclaw/workspace/health-store-miniprogram/server
node index.js &
```

### Q2: 页面空白或无法加载

**原因**：HTTP 服务未启动

**解决**：
```bash
# 检查 HTTP 服务
curl http://localhost:8080/test-page.html

# 如果没响应，重启服务
cd /home/admin/openclaw/workspace/health-store-miniprogram
python3 -m http.server 8080 &
```

### Q3: 签到没反应

**原因**：可能是重复签到或网络问题

**解决**：
1. 查看浏览器控制台（F12）
2. 检查 Network 标签的请求
3. 刷新页面重试

### Q4: 手机访问不了

**原因**：防火墙或 IP 地址错误

**解决**：
```bash
# 获取电脑 IP
ifconfig | grep "inet "

# 用手机访问 http://[IP]:8080/test-page.html
# 确保手机和电脑在同一 WiFi
```
