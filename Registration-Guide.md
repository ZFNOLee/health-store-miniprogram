# 微信小程序账号注册完整指南

## 📋 注册前准备材料

### 企业/个体工商户（推荐）
| 材料 | 要求 |
|------|------|
| 营业执照 | 原件照片或扫描件 |
| 法人身份证 | 正反面照片 |
| 对公账户 | 银行开户许可证 |
| 管理员信息 | 姓名、身份证、手机号 |
| 邮箱 | 未注册过微信开放平台的邮箱 |

### 个人主体（功能受限）
| 材料 | 要求 |
|------|------|
| 身份证 | 正反面照片 |
| 手机号 | 本人实名手机号 |
| 邮箱 | 未注册过微信开放平台的邮箱 |

⚠️ **注意**：个人主体小程序无法开通支付功能，建议使用个体户或公司执照注册。

## 🚀 注册流程（图文步骤）

### 第 1 步：访问注册页面

打开浏览器访问：
```
https://mp.weixin.qq.com
```

点击右上角 **立即注册**

### 第 2 步：选择账号类型

1. 选择 **小程序**
2. 点击 **查看详情** 确认类型
3. 点击 **注册**

### 第 3 步：填写基本信息

```
邮箱：输入未注册过的邮箱
密码：6-32 位，包含字母和数字
验证码：输入邮箱收到的验证码
```

✅ 点击 **注册**

### 第 4 步：邮箱激活

登录刚才的邮箱，点击激活邮件中的链接。

### 第 5 步：信息登记

#### 选择主体类型：
- **企业**（适合公司）
- **个体工商户**（适合个人开店，推荐！）
- **个人**（功能受限，不推荐）

#### 填写主体信息：

**个体工商户示例：**
```
主体名称：XX 市 XX 区 XX 健康咨询店
统一社会信用代码：92XXXXXXXXXXXX
营业执照注册号：XXXXXXXXXXX
营业执照照片：上传清晰照片
```

**法人信息：**
```
姓名：张三
身份证号：XXXXXXXXXXXXXXXXXX
身份证照片：上传正反面
手机号：138XXXXXXXX
短信验证码：输入收到的验证码
```

### 第 6 步：填写管理员信息

```
姓名：（可以是法人，也可以是其他人）
身份证号：XXXXXXXXXXXXXXXXXX
手机号：138XXXXXXXX
职务：总经理/店长
微信绑定：用管理员微信扫码绑定
```

### 第 7 步：填写小程序信息

```
小程序名称：大健康门店管理
（名称需要符合规范，不能包含品牌词、敏感词）

小程序简介：
提供健康产品、积分兑换、会员管理等服务

小程序头像：
上传 144x144 像素的 logo 图片

小程序类目：
选择「商家自营 - 食品/保健」或「医疗健康 - 健康管理」
```

### 第 8 步：对公账户验证（企业/个体户）

微信会向你的对公账户打一笔随机金额（0.01-1 元之间）。

1. 等待 1-3 个工作日收到打款
2. 登录微信后台查看金额
3. 在后台输入收到的金额完成验证

### 第 9 步：等待审核

- 审核时间：1-7 个工作日
- 审核结果会通过微信通知
- 审核通过后会收到小程序 AppID

## ⚠️ 注意事项

### 名称规范
- ❌ 不能包含：中国、中华、国家、微信等
- ❌ 不能包含知名品牌名
- ✅ 可以使用：地区 + 行业 + 店/铺/工作室

### 类目选择
大健康相关可选择的类目：
- 商家自营 - 食品/保健
- 医疗健康 - 健康管理
- 商业服务 - 企业管理

### 审核不通过常见原因
1. 名称涉及敏感词
2. 类目选择错误
3. 营业执照不清晰
4. 经营范围不包含所选类目

## 🎯 注册完成后

拿到小程序账号后，你需要：

### 1. 获取 AppID
登录后台 → **开发** → **开发管理** → **开发设置**
```
AppID(小程序 ID): wxXXXXXXXXXXXXXXXX
```

### 2. 配置项目
修改项目中的 `project.config.json`：
```json
{
  "appid": "wxXXXXXXXXXXXXXXXX"
}
```

### 3. 配置服务器域名
后台 → **开发** → **开发管理** → **开发设置** → **服务器域名**

添加你的后端 API 地址：
```
request 合法域名：https://your-domain.com
```

### 4. 添加开发成员
后台 → **成员管理** → 添加项目成员
- 开发者：可以开发调试
- 体验者：可以扫码体验
